# Carnatic Music Lessons

This repository contains a website for Carnatic Music Lessons designed for ages 6 and up.

## GitHub Pages

This site is deployed using GitHub Pages and can be accessed at:

**URL:** `https://balakrn.github.io/musical-umbrella/`

### Configuration

The site is automatically deployed to GitHub Pages using GitHub Actions workflow (`.github/workflows/pages.yml`).

- **Source**: The `index.html` file in the root of the repository
- **Branch**: Deployed from the `main` branch
- **Workflow**: Triggered on every push to `main` or can be run manually

### Manual Deployment

To manually trigger the deployment:
1. Go to the "Actions" tab in the GitHub repository
2. Select "Deploy to GitHub Pages" workflow
3. Click "Run workflow"

### Setup Instructions

For this GitHub Pages deployment to work, ensure the following settings are configured in the repository:

1. Go to **Settings** â†’ **Pages**
2. Under **Source**, select "GitHub Actions" as the build and deployment source
3. The workflow will automatically deploy the site on the next push to `main`

## Content

The site provides:
- Information about Carnatic Music
- Structured lessons from Sarali Varisai to advanced topics
- Links to audio lessons on Google Drive
- YouTube playlist links for various lessons
- Information about Carnatic music composers
